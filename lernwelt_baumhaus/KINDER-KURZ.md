# 🌳 Baumhaus‑Forscher

## 🎮 So spielst du

# 🪜 An die **Leiter/Liane** stellen + **vorwärts** = **klettern!**
⬆️ hoch ins **Blätterdach** · ⬇️ runter zum **Boden**

# 🦜 Tier antippen = **retten!**
*(rechte Maustaste)*

# 🖼️ **Bilderrahmen** antippen = **Hinweis!**
Schau das **Bild** an: 🍃 Blatt · 💧 Wasser · 🪨 Stein · 🐾 Tier · 💰 Schatz · ➡️ Pfeil

# 🧭 **Kompass** klicken = *Wo bin ich? Wohin?*
*(linke Maustaste)*

# 📷 Kamera klicken = **Foto!**

---

## 🗺️ Die Schnitzeljagd

# 1️⃣ **Bild** im Rahmen anschauen
# 2️⃣ Dem **Pfeil** folgen ➡️⬆️
# 3️⃣ Den **nächsten Rahmen** suchen
# 4️⃣ Den **Schatz** finden! 💰

---

## 🦜 Reit‑Papagei

# Papagei antippen = **aufsteigen**
**W A S D** fliegen · **Leertaste** = **HOCH** ins Blätterdach · **Schleichen** = runter

---

## 🏆 Deine Ziele

# ⬆️ Ganz nach **oben** klettern
# 🧭 **Norden? Osten?** mit dem Kompass finden
# 💰 Den **Schatz** der Schnitzeljagd finden
# 🦜 Viele **Tiere** retten
# ✨ Den **Goldenen Jaguar** entdecken!
