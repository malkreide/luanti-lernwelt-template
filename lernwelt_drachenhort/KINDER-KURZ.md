# 🐉 Drachenhort

## 🎮 So spielst du

# 🥚 Brut‑Ei setzen = **Baby schlüpft!**
*(rechte Maustaste auf den Boden)*

# 🍓 Futter in die Hand, Baby antippen = **füttern!**
**Warte kurz** zwischen dem Füttern – **hab Geduld!**

# 🤝 5‑mal füttern = **Baby wird dein Freund**

# 🐲 Drache antippen = **anfreunden!**
*(rechte Maustaste)*

# 📷 Kamera klicken = **Foto!**
*(linke Maustaste)*

# 🗒️ Tafel antippen = **Aufgabe lösen**
**Eier zählen** · **Farben zuordnen**

# 🐉 Reitdrache antippen = **aufsteigen & fliegen**
**W A S D** fliegen · **Leertaste** hoch · **Schleichen** runter
