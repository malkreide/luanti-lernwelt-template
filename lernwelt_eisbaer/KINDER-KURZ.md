# 🐻‍❄️ Eisbär-Bucht

## 🎮 So spielst du

# 🐾 Tier antippen = **helfen!**
*(rechte Maustaste)*

# 🐧 Wer lebt wo? Pinguin & Robbe am **See**,
# 🦊 Schneehase & Polarfuchs im **Wald**,
# 🐻‍❄️ Eisbär in der **Polarlicht-Höhle!**

# 🗒️ Tafel antippen = **Aufgabe lösen**

# 🐟 Fisch in die Hand → **Futterstelle antippen = füttern!**

# 🌌 Polarlicht-Blöcke setzen = **sie leuchten & wechseln die Farbe!**

# 🎨 Blau-Weiss-Muster bauen (Streifen, Punkte, Schachbrett)

# 🌸 Eisblume setzen = **wächst zu einem Muster!**

# 🛷 Schlitten setzen, antippen = **losfahren!**
*(W/S fahren, A/D lenken)*

# 📷 Kamera + Linksklick = **Tier-Foto fürs Logbuch**
