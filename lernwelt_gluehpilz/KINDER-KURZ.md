# 🍄 Glühpilz‑Wald

## 🎮 So spielst du

# 🦔 Tier antippen = **retten!**
*(rechte Maustaste)*

# 📷 Kamera klicken = **Foto!**
*(linke Maustaste)*

# 🪧 Tafel antippen = **Aufgabe lösen**

# 🌗 **Tag‑Nacht‑Pilz** antippen = **Tag oder Nacht!**
☀️ Tag‑Pilz · 🌅 Dämmer‑Pilz · 🌙 Nacht‑Pilz

# 🍄 Glühpilze leuchten in der **Nacht**

# 🧹 Müll wegklicken = **Wald putzen**

# 🌱 Pilz‑Samen setzen = **Glühpilz wächst**

# 🪲 Leucht‑Käfer antippen = **aufsteigen**
**W A S D** fliegen · **Leertaste** hoch · **Schleichen** runter
