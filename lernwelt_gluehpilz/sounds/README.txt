SOUNDS fuer das Thema "Gluehpilz-Wald"
======================================

Die Rettungs- und Rang-Sounds kommen aus der Engine-Mod "lernwelt"
(Dateien lernwelt_rescue.ogg / lernwelt_rankup.ogg dort im sounds/-Ordner).

Dieses Thema nutzt nur eigene .ogg-Dateien (Mono, kurz) mit GENAU
diesen Namen:

  lernwelt_gluehpilz_kaefer.ogg  -> leiser Summ-/Brumm-Loop (laeuft, solange
                                    man auf dem Leucht-Kaefer reitet)
  lernwelt_gluehpilz_pilz.ogg    -> kurzer, weicher Ton beim Tag-Nacht-Wechsel
                                    (Rechtsklick auf einen Tag-Nacht-Pilz)
  lernwelt_gluehpilz_wald.ogg    -> kurze Wald-Atmo (Vogelzwitschern, Wind)
  lernwelt_gluehpilz_ambient.ogg -> leise Grillen-/Nacht-Atmo (Ambient)

Wichtig:
- Fehlt eine Datei, funktioniert das Thema trotzdem - es kommt nur kein
  Ton (Luanti gibt nur eine Warnung aus).
- Format: Ogg Vorbis (.ogg). Kurze Clips (0.3 - 2 Sekunden) reichen.

Kostenlose Quellen (frei nutzbar, Lizenz beachten):
- https://freesound.org   (z.B. "forest ambience", "cricket", "beetle buzz")
- https://kenney.nl/assets (Kenney Game Assets, viele frei)
- https://opengameart.org

Tipp: Mit Audacity koennt ihr eigene Toene aufnehmen und als .ogg
exportieren - Kinder lieben das.
