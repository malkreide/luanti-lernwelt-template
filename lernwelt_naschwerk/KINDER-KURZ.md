# 🍭 Naschwerk-Tal

## 🎮 So spielst du

# 🔢 Zähl-Naschtisch antippen = **Bonbons zählen!**
*(richtige Zahl anklicken)*

# 🐾 Naschtier antippen = **helfen!**
*(rechte Maustaste)*

# 🍫 Wer lebt wo? Schoko-Fisch im **Fluss**,
# 🍭 Bonbon-Hase im **Lolli-Wald**,
# 🐻 Gummibär auf den **Gummi-Hügeln!**

# 🗒️ Tafel antippen = **Aufgabe lösen** (zählen & Muster)

# 🎨 Muster bauen: **rot – gelb – rot – gelb …**

# 🍬 Bonbon in die Hand → **Naschschale antippen = füttern!**

# 🌈 Regenbogen-Lolli setzen = **er leuchtet & wechselt die Farbe!**

# 🍭 Lolli setzen = **wächst zu einer bunten Süssigkeit!**

# 🛒 Naschwagen setzen, antippen = **losfahren!**
*(W/S fahren, A/D lenken)*

# 📷 Kamera + Linksklick = **Tier-Foto fürs Logbuch**
