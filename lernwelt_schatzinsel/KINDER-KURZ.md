# 🏴‍☠️ Schatzinsel

## 🎮 So spielst du

# 🪙 Goldmünze wegklicken = **zählen!**
*(linke Maustaste)*

# 🧭 Kompass klicken = **wohin? oben/unten?**

# 🗺️ Karte klicken = **wo ist der Weg?**

# 🪧 Wegweiser antippen = **Bild-Hinweis lesen**
*(rechte Maustaste — folge dem Pfeil)*

# 🐠 Tier antippen = **retten!**

# 📷 Kamera klicken = **Foto!**

# 🗒️ Tafel antippen = **Aufgabe lösen**

# ⛵ Floss antippen = **einsteigen**
**W A S D** fahren · **Leertaste** hoch · **Schleichen** runter

## 🏆 Deine Ziele

# 🪙 Viele Goldmünzen zählen
# 🧩 Der Schnitzeljagd zum **Schatz** folgen
# 🧗 Hoch in den **Mastkorb** klettern (oben!)
# 📷 Alle 14 Tiere finden
# 🌟 Den **Goldenen Papagei** finden!
