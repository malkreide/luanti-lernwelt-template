# 🤖 Schrauber-Werkstatt

## 🎮 So spielst du

# 🟥 Schalter setzen, ➖ Leitung, 💡 Lampe = **Schaltung!**
*(Blöcke aneinander auf den Boden)*

# 🖱️ Schalter antippen = **Lampe an, Tür auf!**
**Schalter aus = alles aus.** *(rechte Maustaste)*

# 🧰 Roboter-Bausatz setzen = **Roboter-Rahmen!**

# 🦿 Teile der Reihe nach anbauen:
# **Beine → Körper → Arme → Kopf**
*(Teil in die Hand, Rahmen antippen)*

# 🎢 Auf das Fließband stellen = **es schiebt dich!**

# ♻️ Schrotthaufen abbauen → Recycler antippen = **neues Teil!**

# 🤖 Kaputten Roboter antippen = **reparieren!**
*(rechte Maustaste)*

# 🗒️ Tafel antippen = **Aufgabe lösen**

---

## 🏆 Deine Ziele

# 💡 Eine Lampe zum Leuchten bringen
# 🚪 Eine Tür mit dem Schalter öffnen
# 🤖 Einen Roboter fertig bauen
# 🌟 Den **Meisterbot** finden!
