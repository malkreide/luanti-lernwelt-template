# 🌻 Sonnenhof

## 🎮 So spielst du

# 🌱 Samen setzen = **säen!**
*(rechte Maustaste auf den Boden)*

# 💧 Giesskanne + Rechtsklick aufs Feld = **wachsen lassen!**

# 🌾 Reifen (goldenen) Weizen abbauen = **ernten = Korn!**

# 🏭 Korn in die Hand → Mühle antippen = **Mehl!**
# 🍞 Mehl in die Hand → Mühle antippen = **Brot!**

# 🐄 Tier antippen = **helfen!**
*(rechte Maustaste)*

# 🍽️ Tierfutter in die Hand → Futtertrog antippen = **Tiere pflegen!**

# 🐔 Wer lebt wo? Huhn & Kuh im **Gehege**,
# 🐝 Biene & Maus auf den **Feldern!**

# 🗒️ Tafel antippen = **Aufgabe lösen** (Tiere & Pflanzen)

# 🌻 Sonnenblume setzen = **schöner Hof!**

# 📷 Kamera + Linksklick = **Tier-Foto fürs Logbuch**

---

## 🏆 Deine Ziele

# 🍞 Aus einem Samen ein **Brot** machen
# 🐔 3 Hühnern helfen
# 🍽️ 5 Tiere am Trog pflegen
# 🌟 Das **Goldene Glückshuhn** finden!

# 👈 **Links** = abbauen / ernten / Foto
