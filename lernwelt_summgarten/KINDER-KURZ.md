# 🐝 Summ-Garten

## 🎮 So spielst du

# 🖌️ Pinsel + Rechtsklick auf eine **Blume = bestäuben!**
*(es stäubt gelber Pollen – und du bekommst Nektar)*

# 🌸 Pinsel auf eine **Obstblüte = Frucht wächst!**

# 🍎 Reifen (roten) Apfel abbauen = **ernten!**

# 🍯 Nektar in die Hand → **Honigschleuder** antippen = **Honig!**

# 🔢 **Honigwabe** antippen = **Honigtropfen zählen!**
*(die Wabe füllt sich wieder)*

# 📐 Eine **Wabe ist ein Sechseck** – 6 Ecken!

# 🐞 Tier antippen = **helfen!**
*(rechte Maustaste)*

# 🐝 Wer lebt wo? Biene & Schmetterling auf der **Wiese**,
# 🐞 Marienkäfer & Igel im **Obstgarten!**

# 🗒️ Tafel antippen = **Aufgabe lösen** (Muster, Farben, Zählen)

# 🎨 Bunte Blumen setzen = **schöner Garten!**

# 📷 Kamera + Linksklick = **Tier-Foto fürs Logbuch**

---

## 🏆 Deine Ziele

# 🌸 5 Blumen **bestäuben**
# 🍎 Aus einer Blüte einen **Apfel** wachsen lassen
# 🔢 10 **Honigtropfen** zählen
# 📐 Das **Waben-Muster** an der Tafel fortsetzen
# 🌟 Die **Goldene Honigbiene** finden!
